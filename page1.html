  1. Block/Inline elements

    • i. Block elements take up an entire line and sets the cursor to the next line. Some examples are paragraph or header tags
    • i. Inline elements do not take up an entire line and sets the cursor to the current line. Some examples are the anchor tag or the image tag

  2. Semantic/Non-semantic elements

    • ii. Semantic elements are tags that are used to organize a page for easier accessibility in code and also useful for screen readers. Tags such as header, footer, main.
    • ii. Non-semantic elements are tags that are vague and do not tell what section of the website you are developing. Tags such as div or span.

  3. Forms

    • iii. Forms are used in html to collect user input for things such as a login/password field.
    • iii. The most commonly used form tag is input and it has many types for different purposes such as type text which gets a single line of input from a user.
